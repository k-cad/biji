day09 笔记
python函数的参数传递
    传递方式：
      位置传参
         序列传参
      关键字传参
         字典关键字传参
位置传参：
    实际参数(实参)的对应关系与形式参数的对应关系是按位置来依次对应的
    示意：
        def mymin(a,b,c)
            pass
        
        mymin(1,2,3)
    说明：实参和形参通过位置进行传递和匹配
         实参的个数必须与形参的个数相同
    示例见
序列传参
  序列传参是指在函数调用过程中，用*将序列拆解后按位置进行传递
  说明：序列传参时序列拆解位置将与形参一一对应

关键字传参
  关键字传参是指传参时，按照形参的名称给形参赋值
  形参和实参按名称进行匹配
  说明：实参和形参按形参名进行匹配，可以不按位置进行匹配

字典关键字传参：
  是指实参为字典，将字典用**拆解后进行关键字传参的传参方式
  说明：字典的键名和形参名必须一致
       字典的键名必须为符合标识符命名规则的字符串
       字典的键名必须在形参中存在
函数的综合传参
  1,函数的传参形式，在能确定形参能唯一匹配到相应实参的情况下可以任意组合
  2,函数的位置传参要先于关键字传参
  示例：
      def myfun1(a,b,c)
           pass
      myfun1(100,c=300,b=200)
      myfun1(c=100,b=200,300)#错误
      myfun1(100,*[200,300])
      myfun1(*[100,200],300)
      myfun1(100,*[200],300)
      myfun1(100,**{'c':300,'b':200})
      myfun1(**{'c':300,'b':200})，a=100)
可变 和不可变容器作为实参传递的序列
  可变序列的实参作为实参传递时，在函数内部可以改变容器内的数据
  示例：
      list_as_args.py

函数的缺省参数
  语法：
      def 函数名(形参名1=默认参数1,形参名2=默认参数2,...)
          语句块
    说明：缺省参数必须自右向左依次存在，如果一个参数有缺省参数，
         则其右侧的所有参数都必须有缺省参数
         如：def test_fn(a,b=10,c)#错误示范
         缺省参数可以有0个，1个或全部为缺省参数
  示例：
      default_args.py
      def
函数形参的定义方式
1,位置形参
1,星号元组形参
3,命名关键字形参
4,双星号字典形参
位置形参
语法：def 函数名(形参名1，形参名2，...):
         语句块

星号元组形参
语法：def 函数名(*元组形参名)：
         语句块
作用：
    收集多余的位置参数
说明：
    在一个函数定义中元组形参名最多只有一个，一般命名为'args'
示例

命名关键字形参
  语法：def 函数名(*,命名关键形参1，命名关键形参2，...):
           语句块
        或
        def 函数名(*args,命名关键形参1，命名关键形参2，...):
            语句块
  作用：强制所有的参数都必须用关键字传参或字典关键字传参
  示例：named_keywords_args.py
双星号字典传参
  语法：def 函数名(**字典形参名)
  作用：收集多余的关键字参数
  说明：字典形参名一般命名为‘kwargs'
       一个函数内的字典形参最多只有一个
  示例见：
       star_dict_args.py


函数的参数说明
  位置参数，缺省参数 ，星号元组参数 双星号字典形参可以混合使用

函数参数自左至右的顺序:
1，位置形参
2，星号元组形参
3，命名关键字形参
4，双星号字典形参
示例：
    def fn(a,b=20,*args,c=30,d=40,**kwars):
        pass
    fn(1)
    fn(100,200,300,400,c='C',d='D',e="E")



全局变量 和 局部变量
    局部变量 local varible
        定义在函数内部的变量称为局部变量(函数的形参也是局部变量)
        局部变量只能在函数内部使用
        局部变量在函数被创建时才能被创建，在函数调用后会自动销毁
        当有同名的全局变量和局部变量时，优先访问局部变量
    全局变量 global varible
        定义在函数外部，模块外部的变量称为全局变量
        全局变量，所有函数都可以直接访问(但函数内不能将其直接赋值)
    局部变量说明
        在函数内部赋值语句不会对全局变量造成影响
        局部变量只能在其被声明的函数内部访问，而全局变量可以在整数
        模块范围内访问
